# Морской бой (консольная игра на C++)

«Морской бой» — классическая игра для двух игроков, где участники по очереди называют координаты на сетке 10x10, пытаясь потопить флот противника, состоящий из кораблей разного размера, и первыми уничтожить все корабли соперника.

Учебный проект по дисциплине «Основы программирования».  
Консольная реализация классической игры «Морской бой» на языке C++.

---

##  Описание проекта

Программа представляет собой консольную игру «Морской бой» с возможностью:

- играть вдвоём за одним компьютером  
- играть против компьютера  
- выбирать способ расстановки кораблей (вручную или автоматически)  
- выбирать уровень сложности компьютера  
- вести игру по классическим правилам

Проект демонстрирует работу с:
- функциями  
- классами  
- массивами  
- структурами  
- разбиением проекта на несколько файлов (.h / .cpp)  
- пользовательским вводом  

---

## Возможности игры

- Режим **Игрок против игрока**
- Режим **Игрок против компьютера**
- Ручная расстановка кораблей
- Автоматическая расстановка кораблей
- Два уровня сложности бота:
  - Лёгкий-стреляет случайно  
  - Средний-пытается добивать корабль после попадания  
- Проверка корректности размещения кораблей (корабли не пересекаются и не касаются)
- Защита от повторных выстрелов в одну и ту же клетку

---

## Управление

### Ввод координат
Используется формат:
A1, B5, J10

- Буква-строка (A–J)  
- Число-столбец (1–10)

Пример:
Введите координаты (например A5): B7

---

##  Сборка и запуск

Проект создавался и тестировался в **Visual Studio (C++)**.

Для запуска:
1. Открыть проект в Visual Studio  
2. Собрать решение (Build → Build Solution)  
3. Запустить (F5 или Ctrl+F5)

Дополнительных библиотек не требуется — используется стандартный C++.
